# Welcome to the Agent Control Wiki

Agent Control is a powerful PHP-based platform designed to orchestrate and manage AI agents (powered by Google Jules/Gemini) through GitHub issues. It provides a centralized dashboard for tracking agent activity, managing multiple GitHub accounts, and receiving real-time notifications via webhooks and Telegram.

## Table of Contents

- [Getting Started](Getting-Started.md) - Initial setup and linking your first GitHub account.
- [Managing Projects](Managing-Projects.md) - Linking repositories, project settings, and webhooks.
- [Task Management](Task-Management.md) - Syncing issues, running agents, and status tracking.
- [Issue Templates](Issue-Templates.md) - Creating and using dynamic templates for issue creation.
- [Integrations](Integrations.md) - Configuring Google Jules API and notification channels.
- [Telegram Bot](Telegram-Bot.md) - Mobile interaction and real-time alerts.

## Key Features

- **Multi-Account GitHub Support**: Seamlessly manage repositories across different GitHub accounts.
- **AI Agent Orchestration**: Trigger Google Jules agents to analyze and process GitHub issues.
- **Real-time Feedback**: Agents post comments and status updates directly back to GitHub.
- **Flexible Notifications**: Stay updated via the in-app inbox or your personal Telegram bot.
- **Developer Friendly**: Built with modern tools like Alpine.js, Tailwind CSS, and a robust PHP backend.
